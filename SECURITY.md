# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in the v0 UI Generator extension, please report it privately.

**Do not open a public issue for security vulnerabilities.**

Email: contact@skieart.com

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

We will respond within 48 hours and work with you on a fix before any public disclosure.

## Scope

This policy covers:

- The v0 UI Generator VS Code extension
- The MCP server included with the extension
- API key handling and storage

This policy does not cover:

- The v0 Platform API itself (report to [Vercel](https://vercel.com/security))
- VS Code or other MCP clients
