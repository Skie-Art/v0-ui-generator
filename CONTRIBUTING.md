# Contributing

Thanks for your interest in the v0 UI Generator extension.

## Reporting bugs

Please use the [Bug Report](https://github.com/Skie-Art/v0-ui-generator/issues/new?template=bug_report.yml) template. Include your extension version, environment, and steps to reproduce.

Before filing, check:

- Your v0 API key is valid (`v0_setup_check` passes)
- You're on the latest extension version
- The issue hasn't already been reported

## Suggesting features

Use the [Feature Request](https://github.com/Skie-Art/v0-ui-generator/issues/new?template=feature_request.yml) template. Describe the problem you're solving, not just the solution you want — it helps us find the right approach.

## Code contributions

This repository hosts the extension's public documentation, issue tracker, and marketplace listing. The source code is maintained in a private repository.

If you're interested in contributing code, please open a feature request issue first so we can discuss the approach.

## Code of conduct

Please be respectful and constructive. See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
